# 坦克大战 (Tank Battle)

一个使用纯 HTML5 Canvas + JavaScript 实现的经典坦克大战游戏，无需任何外部依赖。

## 在线试玩

直接用浏览器打开 `index.html` 即可开始游戏。

## 游戏特性

- 经典 90 坦克大战玩法
- 多种地形：砖墙(可破坏)、钢墙(不可破坏)、水域、基地
- 智能敌方坦克 AI
- 多关卡系统，难度递增
- 玩家护盾保护机制
- 爆炸特效与动画
- 生命值与得分系统

## 操作说明

| 按键 | 操作 |
|------|------|
| W / A / S / D | 上 / 左 / 下 / 右 移动 |
| J | 射击 |
| P | 暂停 / 继续 |

## 游戏目标

- 消灭所有敌方坦克进入下一关
- 保护底部的基地（绿色 ★）不被摧毁
- 避免被敌方子弹击中
- 每消灭一辆敌方坦克得 100 分

## 项目结构

```
tank-battle/
├── index.html    # 游戏页面
├── game.js       # 游戏核心逻辑
└── README.md     # 项目说明
```

## 本地运行

```bash
# 直接用浏览器打开
open index.html

# 或使用任意静态服务器
python3 -m http.server 8000
# 然后访问 http://localhost:8000
```

## 技术栈

- HTML5 Canvas
- 原生 JavaScript（无任何依赖）
- CSS3
